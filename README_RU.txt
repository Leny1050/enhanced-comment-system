
### Enhanced Comment System

Спасибо за выбор **Enhanced Comment System**!  
Полную документацию смотрите в `docs/documentation.html`.

**Преимущества**
- Адаптивный интерфейс (светлая и тёмная темы)
- Простая интеграция (одна строка кода)
- Модерирование, панель администратора
- Защита от спама (reCAPTCHA v3)
- Чистый PHP (готов к миграции на Laravel)
- Совместим с MySQL 8 / MariaDB
