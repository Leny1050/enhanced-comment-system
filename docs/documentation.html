Enhanced Comment System

Overview

Enhanced Comment System is a modern, responsive and secure commenting solution you can embed into any PHP-based website. It offers a sleek UI, spam protection, and simple integration (copy-paste one code snippet).

Quick Start (Local XAMPP)

  1. Unzip the package into C:/xampp/htdocs/comment_system.
  2. Create a MySQL database comments (utf8mb4) and import install.sql.
  3. Open config/config.php and replace all <<<CHANGE-ME>>> placeholders (DB credentials, admin password hash, reCAPTCHA keys).
  4. Visit http://localhost/comment_system/public/index.php – the widget will load.
  5. Moderate comments at http://localhost/comment_system/public/admin.php.

Deploying on Shared / VPS Hosting

  1. Upload the comment_system folder to your public_html (or sub‑folder).
  2. Create a MySQL database from your hosting control panel; note credentials.
  3. Edit config/config.php with these credentials.
  4. Import install.sql via phpMyAdmin (or MySQL CLI).
  5. Make sure public/ is accessible via URL. You can point a sub‑domain to that folder or move files up one level.
  6. Enable HTTPS on your host (Let's Encrypt) – required for reCAPTCHA.

Customization Tips

  • Change the logo at public/assets/img/logo.png.
  • Edit primary colors in public/assets/css/style.css – variables at the top.
  • Switch default theme (light/dark) in config/config.php.
  • Add social login by creating config/oauth.php and extending CommentController.php.

Обзор

Enhanced Comment System – современная, адаптивная и безопасная система комментариев, которую можно встроить на любой PHP‑сайт. Стильный интерфейс, защита от спама и простая интеграция (копируйте один сниппет).

Быстрый старт (XAMPP)

  1. Распакуйте архив в C:/xampp/htdocs/comment_system.
  2. Создайте базу MySQL comments (utf8mb4) и импортируйте install.sql.
  3. Откройте config/config.php и замените все пометки <<<CHANGE-ME>>> (данные БД, хеш пароля админа, ключи reCAPTCHA).
  4. Перейдите по адресу http://localhost/comment_system/public/index.php – виджет загрузится.
  5. Панель модерации доступна на http://localhost/comment_system/public/admin.php.

Развёртывание на хостинге

  1. Загрузите папку comment_system на public_html вашего хостинга.
  2. Создайте базу данных MySQL и сохраните реквизиты.
  3. Пропишите их в config/config.php.
  4. Через phpMyAdmin импортируйте файл install.sql.
  5. Убедитесь, что директория public/ открывается по URL (если нужно, переместите файлы на уровень выше).
  6. Включите HTTPS (Let’s Encrypt) – требуется для reCAPTCHA.

Советы по кастомизации

  • Замените логотип: public/assets/img/logo.png.
  • Измените основные цвета в public/assets/css/style.css – переменные CSS в начале файла.
  • Установите тему по умолчанию (light/dark) в config/config.php.
  • Добавьте социальный вход, создав config/oauth.php и расширив CommentController.php.
